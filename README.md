# SportNest – Sports Facility Booking Platform

A full-stack sports facility booking platform built with the MERN stack, where users can explore sports venues like football turfs, badminton courts, swimming pools, and tennis courts, book available time slots, and manage their own facility listings.

## Features

* Fully responsive design (Mobile, Tablet, Desktop)
* Modern and clean UI
* Protected facility details & booking route
* User authentication system (JWT + HTTPOnly cookies)
* Google login support
* Register and login functionality
* Persistent Navbar & Footer
* Search facilities by name
* Filter facilities by sport type
* Dynamic facility rendering from MongoDB database
* Toast notifications for success/error messages
* Loading spinner while fetching data
* Not-found page implementation
* Add Facility (Create) feature
* Manage My Facilities (Update/Delete) feature
* My Bookings page with cancel booking option
* Image upload support via imgbb
* Dark / Light theme toggle
* Smooth user experience with animations

## How to Run Locally

* Clone the repository:
git clone https://github.com/rakibmur420-source/B13-assignment-9_client

* Go to the project folder:
cd sportnest-client

* Install dependencies:
npm install

* Run the development server:
npm run dev

* Environment Variables (.env)
VITE_IMGBB_API_KEY=
VITE_FIREBASE_API_KEY=
